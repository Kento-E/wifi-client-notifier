# 貢献ガイド

WiFi Client Notifierへの貢献を検討いただき、ありがとうございます！

## 貢献の方法

### バグ報告

バグを発見した場合は、以下の情報を含めてIssueを作成してください:

- **WiFiルータのモデル名**: 例: WG2600HP4
- **ファームウェアバージョン**: ルータの管理画面で確認できます
- **実行環境**: OS、Pythonバージョン
- **エラーメッセージ**: 完全なエラーメッセージとスタックトレース
- **ログファイル**: `log_level: "DEBUG"`で取得したログ
- **再現手順**: 問題を再現する手順

### 機能リクエスト

新機能の提案は大歓迎です！Issueで以下を説明してください:

- 提案する機能の説明
- ユースケース（なぜその機能が必要か）
- 可能であれば、実装のアイデア

### Pull Request

#### 1. ローカル開発環境のセットアップ

```bash
# リポジトリをフォーク後、クローン
git clone https://github.com/YOUR_USERNAME/wifi-client-notifier.git
cd wifi-client-notifier

# 依存パッケージをインストール
pip install -r requirements.txt

# 開発ブランチを作成
git checkout -b feature/your-feature-name
```

#### 2. コーディング規約

- **PEP 8**: Pythonコードスタイルガイドに従う
- **型ヒント**: 可能な限り型ヒントを使用
- **ドキュメント文字列**: すべての関数にdocstringを追加
- **日本語コメント**: コメントは日本語でも英語でもOK

例:
```python
def parse_device_list(html_content: str) -> List[Dict[str, str]]:
    """
    HTMLからデバイスリストを抽出する
    
    Args:
        html_content: ルータから取得したHTML
        
    Returns:
        デバイス情報の辞書のリスト
    """
    # 実装...
```

#### 3. テスト

- 変更をテストして、既存の機能を壊していないことを確認
- 可能であれば、新機能にはテストを追加

#### 4. コミット

```bash
# 変更をステージング
git add .

# わかりやすいコミットメッセージで コミット
git commit -m "Add support for WG2600HP4 router model"

# プッシュ
git push origin feature/your-feature-name
```

#### 5. Pull Requestを作成

- フォークしたリポジトリからPull Requestを作成
- 変更内容を明確に説明
- **関連するIssueがあれば、PR説明に`Closes #番号`を含めてリンク**
  - 例: `Closes #10` でイシュー#10をPRマージ時に自動クローズ
  - 複数のイシューの場合: `Closes #10, #15`
  - バグ修正の場合は `Fixes #番号` も使用可能

## 特に歓迎する貢献

### ルータモデル対応

異なるWiFiルータモデルの対応情報は大変貴重です:

1. ルータモデル名とファームウェアバージョン
2. ログインエンドポイントとパラメータ
3. デバイスリスト取得エンドポイント
4. レスポンス形式（HTML/JSON）のサンプル
5. 動作確認済みのカスタマイズコード

### ドキュメント改善

- タイポの修正
- 説明の改善
- 他言語への翻訳
- より詳しい使用例

### コードの改善

- バグ修正
- パフォーマンスの向上
- エラーハンドリングの改善
- コードの可読性向上

## ルータモデル対応情報の共有方法

あなたのWiFiルータで動作確認ができた場合、以下のテンプレートでIssueを作成してください:

```markdown
## 動作確認済みルータ

**モデル**: WG2600HP4
**ファームウェア**: 1.2.3

### 必要なカスタマイズ

なし（デフォルト設定で動作）

または

```python
# login()メソッドを以下のように変更
def login(self):
    # カスタマイズコード
```

### デバイスリスト取得エンドポイント

`http://192.168.10.1/index.cgi/wireless_client_list`

### レスポンス形式

JSON / HTML（どちらか記載）

### 備考

特記事項があれば記載
```
